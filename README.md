# ENERTROL S.A - E-commerce Website

Sitio web de comercio electrónico para **ENERTROL S.A** (Energía y Control), empresa especializada en sistemas auxiliares de energía eléctrica.

## Acerca de ENERTROL S.A

Energía y Control, ENERTROL S.A. se crea en el año 2004 ante la necesidad identificada en el mercado de un integrador para sistemas auxiliares de energía.

**Misión:** "Manejo Seguro y Confiable de la Energía Eléctrica, adaptándola a su Medida"

## Tecnologías Utilizadas

- **React 19** - Librería de interfaz de usuario
- **React Router DOM** - Navegación entre páginas
- **Vite** - Herramienta de construcción rápida
- **CSS3** - Estilos personalizados

## Instalación y Desarrollo

```bash
# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Vista previa de la construcción
npm run preview

# Ejecutar linter
npm run lint
```

## Estructura del Proyecto

```
src/
├── components/       # Componentes reutilizables (Navbar, Footer)
├── pages/           # Páginas principales
│   ├── Home.jsx     # Página de inicio
│   ├── QuienesSomos.jsx  # Página "Quiénes Somos"
│   ├── Productos.jsx     # Catálogo de productos
│   └── Contacto.jsx      # Formulario de contacto
├── App.jsx          # Componente principal con rutas
└── main.jsx         # Punto de entrada
```
